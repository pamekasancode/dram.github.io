Cara Bermain Di Computer

Gunakan keyboard kalian untuk memainkannya

Berikut kode keyboard:

A : Open

S : Close

D : Crash

F : Tom

G : Bass

Space : Snare

Untuk memutar music, klik icon music lalu pilih file music yang kalian mau!
